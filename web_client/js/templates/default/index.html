Welcome to the Online Oil Database (BETA)

This is a web based interface for managing a database of oils. With it, you can:

  • Search for oils using a wide range of criteria.
  • View individual oil properties.
  • Add and modify records (assuming you have the access rights).
  • Analyze and compare sets of oils.

This software was developed by the NOAA Office of Response and Restoration (OR&R) Emergency Response Division.
